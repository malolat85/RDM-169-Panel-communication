# RDM-169-Panel-communication

Arduino read command beetween Blaupunkt RDM 169 Munchen and front panel.<br>
Not all commands are known, but basic LCD and buttons are recognised :)<br>
<br>
Connection on front panel:<br>
PIN 6 - CLK<br>
PIN 7 - DATA<br>
PIN 8 - ENABLE<br>
<br>
Arduino Pins:<br>
10 - ENABLE<br>
11 - DATA<br>
13 - CLK
